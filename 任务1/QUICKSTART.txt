📱 DeepSeek 余额监控小组件 - 快速使用指南

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔧 功能说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 安卓系统级桌面小组件
- 不打开 App 就能直接看到余额
- 后台自动定时刷新（默认 30 分钟）
- 点击余额数字打开 App 配置
- 点击状态文字手动刷新

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📥 安装步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 在浏览器中下载 APK
② 点击下载好的 APK 文件
③ 若提示"未知来源"：
   设置 → 安全 → 允许安装未知应用
   （或在弹窗中点"设置"去授权）
④ 等几秒安装完成，桌面会出现"DeepSeek 余额"图标

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔑 配置 API Key
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 打开"DeepSeek 余额"App
② 访问 https://platform.deepseek.com
   注册账号 → 创建 API Key → 复制
③ 回到 App，粘贴 Key 到输入框
④ 点"测试连接"看到 ✓ 后保存

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📲 添加桌面小组件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 长按桌面空白处
② 选"小组件"或"Widgets"
③ 找到"DeepSeek 余额"
④ 拖到桌面（推荐 2x1 或 4x2 尺寸）
⑤ 完事！以后不用打开 App 就能看余额了

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 日常使用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 看余额：直接在桌面小组件看
• 改配置：点余额数字
• 手动刷新：点"在线"或"离线"状态
• 删小组件：长按拖到垃圾桶

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️ 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❓ 找不到小组件？
   → 试试桌面长按 → 小部件 → 搜索"DeepSeek"

❓ 余额不更新？
   → 设置 → 电池 → DeepSeek 余额 → 不优化

❓ 一直显示"!"离线？
   → 打开 App 测一下，可能是 Key 错了
